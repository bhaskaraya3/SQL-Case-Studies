# Retail Superstore Sales Analysis – SQL Case Study

This project analyzes retail sales data using SQL
to evaluate revenue performance, customer behavior,
product portfolio efficiency, and delivery operations.

## Business Questions Answered
• What drives revenue growth across categories and regions?
• Which customer segments generate high-value orders?
• How does delivery speed impact revenue?
• Where are operational inefficiencies present?
• Is revenue concentrated among few customers?

## Skills Demonstrated
• SQL data cleaning  
• Exploratory & business analysis  
• Advanced SQL (CTEs, subqueries)  
• Operational & revenue insights
